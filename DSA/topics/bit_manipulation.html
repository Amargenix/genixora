    ‚ö° Important Bit Manipulation Tricks

    ‚úÖ 1. Check if a Number is Even or Odd

    Use the **AND operator**:

    
                bool isEven(int n) {
                    return (n & 1) == 0; // If last bit is 0, the number is even
                }

    ‚úÖ 2. Check if a Number is a Power of 2

    Only **one bit** is set in powers of 2.

    
                bool isPowerOfTwo(int n) {
                    return (n & (n - 1)) == 0 && n > 0;
                }

    ‚úÖ 3. Count the Number of 1s in Binary (Hamming Weight)

    Uses Brian Kernighan‚Äôs Algorithm.

    
                int countSetBits(int n) {
                    int count = 0;
                    while (n) {
                        n = n & (n - 1); // Removes the rightmost set bit
                        count++;
                    }
                    return count;
                }

    ‚úÖ 4. Reverse Bits of a Number

    
                uint32_t reverseBits(uint32_t n) {
                    uint32_t result = 0;
                    for (int i = 0; i < 32; i++) {
                        result = (result << 1) | (n & 1);
                        n = n >> 1;
                    }
                    return result;
                }

    ‚úÖ 5. Swap Two Numbers Without Using a Third Variable

    
                void swap(int &a, int &b) {
                    a = a ^ b;
                    b = a ^ b;
                    a = a ^ b;
                }
