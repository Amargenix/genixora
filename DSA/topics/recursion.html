    **Recursion** is a technique where a **function calls itself** to solve a problem by breaking it down into **smaller subproblems**.

    üîÅ How Recursion Works?

    Recursion follows the **Divide & Conquer approach**, solving problems in **two steps:**

    • ‚úÖ **Base Case** - The condition where recursion stops.
    • ‚ôªÔ∏è **Recursive Case** - The function calls itself with a smaller problem.

    üõ†Ô∏è Example: Factorial of a Number

                    if (n == 0) return 1; // Base Case
                    return n * factorial(n - 1); // Recursive Case

    üîÑ Types of Recursion

    üîπ 1. Direct Recursion

    A function calls itself directly.

                    func();  // Direct self-call
                }

    üî∏ 2. Indirect Recursion

    Two or more functions call each other.

    
                void A() { B(); }
                void B() { A(); }

    üîÑ 3. Tail Recursion

    Recursive call is the **last statement** in the function.

    
                void tailRecursion(int n) {
                    if (n == 0) return;
                    tailRecursion(n - 1);
                }

    üîÉ 4. Non-Tail Recursion

    Recursive call is **not the last statement**.

    
                int nonTail(int n) {
                    if (n == 0) return 0;
                    return n + nonTail(n - 1);
                }
