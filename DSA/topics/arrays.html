    Arrays in Data Structures

    An **Array** is a **linear data structure** that stores a **fixed-size** sequence of elements of the **same data type**. Arrays allow **efficient access** to elements using an index.

    🚀 Why Use Arrays?

    • 💡 **Efficient Element Access** - Access elements in **O(1) time** using indexing.
    • ⚡ **Fixed Size** - Memory is **preallocated**, avoiding dynamic memory issues.
    • 📊 **Easier Data Organization** - Suitable for **matrices, tables, and graphs**.
    • 🔍 **Used in Searching & Sorting Algorithms** - Binary Search, Merge Sort, Quick Sort, etc.

    📖 Types of Arrays

    📌 1. One-Dimensional Arrays

    A **1D array** stores elements **linearly** in a single row.

    
                int arr[5] = {1, 2, 3, 4, 5};

    📌 2. Multi-Dimensional Arrays (2D & 3D)

    Multi-dimensional arrays allow storing **tabular data** (like matrices).

    
                int matrix[3][3] = {{1,2,3}, {4,5,6}, {7,8,9}};

    📌 3. Dynamic Arrays

    Unlike static arrays, **dynamic arrays** can grow or shrink in size.

    
                vector arr;  // C++ Dynamic Array (STL)
                arr.push_back(10);

    🔄 Common Array Operations

    📌 1. Traversal (Accessing Elements)

    Iterating through an array using a loop:

    
                for(int i = 0; i < 5; i++) {
                    cout << arr[i] << " ";
                }

    📌 2. Insertion

    Adding an element requires shifting elements (O(n) time complexity).

    
                int arr[6] = {1, 2, 3, 4, 5}; 
                int pos = 2, value = 10;
                for (int i = 5; i >= pos; i--) {
                    arr[i] = arr[i - 1];
                }
                arr[pos] = value;

    📌 3. Deletion

    Removing an element also requires shifting (O(n) time complexity).

    
                int pos = 2;
                for (int i = pos; i < 4; i++) {
                    arr[i] = arr[i + 1];
                }

    📌 4. Searching (Linear Search & Binary Search)

    Finding an element in an array.

    🔍 Linear Search (O(n))

    
                int search(int arr[], int n, int key) {
                    for(int i = 0; i < n; i++) {
                        if(arr[i] == key) return i;
                    }
                    return -1;
                }

    ⚡ Binary Search (O(log n))

    
                int binarySearch(int arr[], int left, int right, int key) {
                    while (left <= right) {
                        int mid = left + (right - left) / 2;
                        if (arr[mid] == key) return mid;
                        if (arr[mid] < key) left = mid + 1;
                        else right = mid - 1;
                    }
                    return -1;
                }

    📌 5. Sorting (Bubble Sort & Quick Sort)

    📊 Bubble Sort (O(n²))

    
                void bubbleSort(int arr[], int n) {
                    for(int i = 0; i < n-1; i++) {
                        for(int j = 0; j < n-i-1; j++) {
                            if(arr[j] > arr[j+1]) swap(arr[j], arr[j+1]);
                        }
                    }
                }

    ⚡ Quick Sort (O(n log n))

    
                int partition(int arr[], int low, int high) {
                    int pivot = arr[high];
                    int i = low - 1;
                    for (int j = low; j < high; j++) {
                        if (arr[j] < pivot) swap(arr[++i], arr[j]);
                    }
                    swap(arr[i + 1], arr[high]);
                    return i + 1;
                }
            
                void quickSort(int arr[], int low, int high) {
                    if (low < high) {
                        int pi = partition(arr, low, high);
                        quickSort(arr, low, pi - 1);
                        quickSort(arr, pi + 1, high);
                    }
                }

    📊 Time Complexity Analysis of Array Operations

    Operation Best Case Worst Case
    Access O(1) O(1)
    Search (Linear) O(1) O(n)
    Search (Binary) O(1) O(log n)
    Insertion O(1) O(n)
    Deletion O(1) O(n)

    🌍 Real-World Applications of Arrays

    • 🛒 **E-commerce websites** (e.g., Amazon) use arrays for product catalogs.
    • 📅 **Calendars and scheduling apps** use arrays for storing dates & events.
    • 🎮 **Game development** (e.g., 2D/3D graphics use arrays for pixel data).
    • 🚗 **Navigation systems** use arrays for GPS tracking and route optimization.
