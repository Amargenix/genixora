    1. Introduction to DevOps

    What is DevOps?

    DevOps is a cultural and technical movement that bridges software development (Dev) and IT operations (Ops) through:

    • ✅ Continuous Integration & Delivery
    • ✅ Infrastructure as Code (IaC)
    • ✅ Monitoring & Logging
    • ✅ Collaboration & Automation

    DevOps Lifecycle

    Phase Activities Tools
    Plan Requirements, Tracking Jira, Trello
    Code Development, Review Git, GitHub
    Build Compilation, Packaging Maven, Gradle
    Test Automated Testing Selenium, JUnit
    Deploy Release Management Ansible, Kubernetes
    Operate Infrastructure Management Terraform, AWS
    Monitor Performance Tracking Prometheus, Grafana
