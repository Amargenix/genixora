    Cloud Computing in Software Engineering

    1. Introduction to Cloud Computing

    Cloud computing delivers on-demand computing services over the internet, including:

    • ✅ Servers (Compute power)
    • ✅ Storage (Files, databases)
    • ✅ Networking (CDNs, load balancers)
    • ✅ Software (SaaS applications)

    Why Cloud for Software Engineering?

    • Scalability: Handle traffic spikes automatically
    • Cost-Efficiency: Pay only for what you use
    • Global Availability: Deploy apps closer to users
    • Faster Development: Pre-built cloud services

    2. Cloud Service Models

    IaaS (Infrastructure as a Service)

    Provides: Virtual machines, storage, networking

    You Manage: OS, middleware, applications

    Examples: AWS EC2, Azure VMs, Google Compute Engine

    PaaS (Platform as a Service)

    Provides: Runtime environment, development tools

    You Manage: Applications and data

    Examples: Heroku, AWS Elastic Beanstalk, Google App Engine

    SaaS (Software as a Service)

    Provides: Ready-to-use applications

    You Manage: Nothing (just use it)

    Examples: Gmail, Salesforce, Slack

    3. Cloud Deployment Models

    Model Description Use Cases Examples
    Public Cloud Shared infrastructure, pay-as-you-go Startups, Web apps AWS, Azure, GCP
    Private Cloud Dedicated infrastructure for one organization Banks, Government OpenStack, VMware
    Hybrid Cloud Mix of public + private clouds Enterprise apps with sensitive data AWS Outposts, Azure Stack
    Multi-Cloud Using multiple cloud providers Avoid vendor lock-in AWS + Azure + GCP

    4. Major Cloud Providers

    AWS (Amazon Web Services)

    • Market leader (33% share)
    • 200+ services
    • Strong in IaaS and PaaS
    • Key Services: EC2, S3, Lambda, RDS

    Microsoft Azure

    • Enterprise focus
    • Deep Windows integration
    • Strong hybrid cloud
    • Key Services: VMs, Azure Functions, SQL DB

    Google Cloud (GCP)

    • Strong in AI/ML
    • Excellent data analytics
    • Global network
    • Key Services: BigQuery, Kubernetes Engine
