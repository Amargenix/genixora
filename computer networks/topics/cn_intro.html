Computer Networks

    Introduction to Computer Networks

    What is a Computer Network?

    A computer network is a system that connects multiple computers and other devices (like printers, servers, etc.) to allow communication and resource sharing between them. Networks enable users to share files, access remote resources, and communicate efficiently.

    Key Components of a Network

    • Nodes: Devices connected to the network (computers, printers, servers)
    • Links: The physical or wireless connections between nodes
    • Protocols: Rules governing data communication (TCP/IP, HTTP, FTP)
    • Network Interface Card (NIC): Hardware that connects a device to a network
    • Switches and Routers: Devices that manage data traffic

    Types of Networks

    Type Description Coverage
    PAN (Personal Area Network) Connects devices around an individual 10 meters
    LAN (Local Area Network) Connects devices in a small geographical area Building or campus
    MAN (Metropolitan Area Network) Covers a city or large campus 5-50 km
    WAN (Wide Area Network) Spans large geographical areas (countries, continents) Worldwide

    Network Topologies

    Network topology refers to the arrangement of different elements in a computer network:

    • Bus: All devices connected to a single cable
    • Star: All devices connected to a central hub/switch
    • Ring: Devices connected in a circular fashion
    • Mesh: Devices interconnected with multiple paths
    • Tree: Hierarchical combination of star and bus topologies
    • Hybrid: Combination of two or more topologies

    Network Protocols

    Protocols are sets of rules that govern how devices communicate on a network:

    TCP/IP

    Transmission Control Protocol/Internet Protocol - the fundamental protocol suite for the Internet

    HTTP/HTTPS

    Hypertext Transfer Protocol (Secure) - for web browsing

    FTP

    File Transfer Protocol - for file transfers

    SMTP/POP/IMAP

    Protocols for email communication

    OSI Model

    The Open Systems Interconnection (OSI) model describes seven layers of network communication:

    1. Physical Layer: Transmits raw bit stream over physical medium
    2. Data Link Layer: Provides node-to-node data transfer
    3. Network Layer: Handles routing and forwarding
    4. Transport Layer: Ensures end-to-end communication
    5. Session Layer: Manages connections between applications
    6. Presentation Layer: Handles data translation and encryption
    7. Application Layer: Interface for user applications

    Importance of Computer Networks

    • Resource sharing (files, printers, applications)
    • Improved communication (email, instant messaging, video conferencing)
    • Data centralization and backup
    • Cost efficiency through shared resources
    • Remote access to information
    • Collaboration between users
    • Business process automation

    • 5G networks for faster wireless communication
    • Software-Defined Networking (SDN)
    • Network Function Virtualization (NFV)
    • Internet of Things (IoT) networks
    • Edge computing and fog networking
    • Artificial Intelligence in network management
    • Quantum networking
