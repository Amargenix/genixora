    Wireless Computer Networks

    Technologies, Standards, and Implementation

    Introduction to Wireless Networking

    Wireless networks enable devices to connect without physical cables, using radio waves, microwaves, or infrared signals for communication. They provide mobility, flexibility, and scalability compared to wired networks.

    Key Advantages

    • Mobility: Access network resources from anywhere within coverage
    • Installation Speed: Quicker to deploy than wired networks
    • Flexibility: Easy to modify and expand
    • Cost-Effective: Reduces cabling infrastructure costs
    • Scalability: Easily accommodate new users

    Challenges

    • Security: More vulnerable to attacks
    • Interference: Susceptible to signal disruption
    • Bandwidth: Generally lower than wired
    • Coverage: Limited range and dead zones
    • Reliability: More prone to environmental factors

    Wireless Network Types

    WPAN

    Wireless Personal Area Network

    • Range: Up to 10 meters
    • Examples: Bluetooth, Zigbee, NFC
    • Use Cases: Wearables, peripherals, smart home
    • Standards: IEEE 802.15

    WLAN

    Wireless Local Area Network

    • Range: Up to 100 meters
    • Examples: Wi-Fi, Hotspots
    • Use Cases: Home/office networks, public access
    • Standards: IEEE 802.11

    WMAN

    Wireless Metropolitan Area Network

    • Range: Several kilometers
    • Examples: WiMAX, LTE
    • Use Cases: City-wide networks, ISP last-mile
    • Standards: IEEE 802.16

    WWAN

    Wireless Wide Area Network

    • Range: Global coverage
    • Examples: Cellular (3G, 4G, 5G), Satellite
    • Use Cases: Mobile phones, IoT, remote access
    • Standards: GSM, CDMA, LTE, 5G NR

    Ad Hoc Networks

    Device-to-Device Connections

    • Types: MANET, VANET, WSN
    • Examples: Mesh networks, vehicular networks
    • Use Cases: Emergency services, IoT, military
    • Features: Self-configuring, infrastructure-less

    Key Wi-Fi Technologies

    MIMO

    Multiple Input Multiple Output - uses multiple antennas to improve performance

    MU-MIMO

    Multi-User MIMO - allows simultaneous data transmission to multiple devices

    OFDMA

    Orthogonal Frequency Division Multiple Access - improves efficiency in Wi-Fi 6

    Beamforming

    Focuses signal toward specific devices rather than broadcasting in all directions

    TWT

    Target Wake Time - improves battery life for IoT devices (Wi-Fi 6)

    WPA3

    Latest Wi-Fi security protocol with stronger encryption

    Wireless Network Components

    Access Points (AP)

    Central hub that connects wireless devices to a wired network

    Types: Standalone, Controller-based, Cloud-managed

    Wireless Routers

    Combines AP, router, switch and often modem functionality

    Features: NAT, Firewall, DHCP

    Wireless Controllers

    Centralized management for multiple APs in enterprise networks

    Functions: Configuration, Monitoring, Security

    Wireless NICs

    Network Interface Cards that enable devices to connect wirelessly

    Forms: PCIe, USB, M.2, Onboard

    Range Extenders

    Devices that amplify wireless signals to expand coverage

    Alternatives: Mesh systems, Powerline adapters

    Wireless Bridges

    Connects two wired networks wirelessly

    Use Cases: Building-to-building connections

    Antennas

    Components that transmit and receive radio waves

    Types: Omnidirectional, Directional, Sector

    Wireless Security

    Common Threats

    • Eavesdropping: Unauthorized monitoring of traffic
    • Rogue APs: Unauthorized access points
    • Man-in-the-Middle: Intercepting communications
    • DoS Attacks: Disrupting network availability
    • Evil Twin: Fake access points
    • Packet Sniffing: Capturing unencrypted data

    Security Measures

    • Encryption: WPA3, WPA2, AES
    • Authentication: 802.1X, RADIUS
    • MAC Filtering: Allow only known devices
    • SSID Management: Hide SSID, change defaults
    • Firewalls: Network and host-based
    • VPN: Secure remote access

    Wi-Fi Security Protocol Evolution

    Protocol Year Encryption Key Features Vulnerabilities
    WEP 1997 RC4 (64/128-bit) First security protocol Easily cracked, deprecated
    WPA 2003 TKIP/RC4 Temporal Key Integrity Protocol Still vulnerable to attacks
    WPA2 2004 AES-CCMP Strong encryption, mandatory KRACK attack vulnerability
    WPA3 2018 AES-GCMP-256 Simultaneous Auth, Forward Secrecy Dragonblood (mitigated)
