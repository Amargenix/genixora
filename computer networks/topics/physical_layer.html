Computer Networks

    Physical Layer

    Introduction to Physical Layer

    The Physical Layer is the first and lowest layer in the OSI model of computer networking. It deals with the physical connection between devices and the transmission and reception of raw bit streams over a physical medium.

    This layer defines the electrical, mechanical, procedural, and functional specifications for activating, maintaining, and deactivating the physical link between network devices.

    Key Responsibilities

    • Bit representation and synchronization
    • Physical topology definition
    • Transmission mode determination
    • Physical medium characteristics
    • Data rate control
    • Line configuration

    Physical Layer Functions

    Bit Representation

    Defines how bits are converted to signals (electrical, optical, or radio waves).

    • Encoding schemes (NRZ, Manchester, 4B/5B)
    • Modulation techniques (ASK, FSK, PSK)
    • Signal voltage levels
    • Bit synchronization

    Data Rate

    Determines the transmission rate (number of bits sent per second).

    • Bandwidth limitations
    • Nyquist Theorem
    • Shannon's Capacity
    • Baud rate vs bit rate

    Physical Topology

    Defines how devices are physically connected.

    • Bus, Star, Ring, Mesh
    • Point-to-point vs multipoint
    • Physical vs logical topology
    • Network layout considerations

    Transmission Mode

    Determines the direction of data flow.

    • Simplex (one direction)
    • Half-duplex (both directions, not simultaneously)
    • Full-duplex (both directions simultaneously)
    • Parallel vs serial transmission

    Line Configuration

    Defines the connection type between devices.

    • Point-to-point (direct connection)
    • Multipoint (shared connection)
    • Master-slave relationships
    • Peer-to-peer connections

    Physical Medium

    Specifies the characteristics of the transmission medium.

    • Guided (wired) vs unguided (wireless)
    • Attenuation and distortion
    • Noise and interference
    • Propagation characteristics

    Transmission Media

    Twisted Pair Cable

    Two insulated copper wires twisted together to reduce interference.

    Types:

    • UTP (Unshielded Twisted Pair)
    • STP (Shielded Twisted Pair)

    Categories: Cat5, Cat5e, Cat6, Cat6a, Cat7, Cat8

    Applications: Ethernet networks, telephone lines

    Coaxial Cable

    Central conductor surrounded by insulation, shield, and outer jacket.

    Types:

    • RG-6 (for cable TV)
    • RG-58 (for thin Ethernet)
    • RG-59 (for CCTV)

    Applications: Cable TV, broadband internet, CCTV

    Fiber Optic Cable

    Glass or plastic fibers carrying light signals.

    Types:

    • Single-mode fiber (long distance)
    • Multi-mode fiber (short distance)

    Applications: Backbone networks, FTTx, data centers

    Wireless Media

    Uses electromagnetic waves for transmission.

    Types:

    • Radio waves (WiFi, Bluetooth)
    • Microwave (point-to-point links)
    • Infrared (short-range communication)
    • Satellite (global coverage)

    Applications: Mobile networks, WiFi, satellite comms

    Physical Layer Protocols and Standards

    Ethernet Standards

    • 10BASE-T (10 Mbps over UTP)
    • 100BASE-TX (Fast Ethernet)
    • 1000BASE-T (Gigabit Ethernet)
    • 10GBASE-T (10 Gigabit Ethernet)
    • 40GBASE-T and 100GBASE-T

    Wireless Standards

    • IEEE 802.11 (WiFi)
    • Bluetooth (IEEE 802.15.1)
    • Zigbee (IEEE 802.15.4)
    • LTE/5G cellular
    • NFC (Near Field Communication)

    Optical Standards

    • SONET/SDH
    • Fiber Channel
    • DWDM (Dense Wavelength Division Multiplexing)
    • FTTx (Fiber to the x)

    Other Standards

    • RS-232 (serial communication)
    • USB (Universal Serial Bus)
    • HDMI (High-Definition Multimedia Interface)
    • Powerline Communication

    Hub

    📶

    Modem

    Modulates/demodulates signals for transmission

    📡

    Access Point

    Wireless base station for network access

    🔀

    Converts between different media types
