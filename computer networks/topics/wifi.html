    Wi-Fi Technology in Computer Networks

    Wireless Networking Standards, Technologies, and Security

    What is Wi-Fi?

    Wi-Fi (Wireless Fidelity) is a family of wireless network protocols based on the IEEE 802.11 standards, commonly used for local area networking of devices and Internet access. Wi-Fi allows electronic devices to connect to a wireless LAN (WLAN) network, mainly using the 2.4 GHz and 5 GHz radio bands.

    Key Characteristics

    • Wireless connectivity without cables
    • Uses radio frequency technology
    • Standardized by IEEE 802.11 family
    • Operates in unlicensed spectrum
    • Typical range: 20m indoors to 100m outdoors

    Main Advantages

    • Mobility and convenience
    • Easy installation and scalability
    • Cost-effective (no cabling infrastructure)
    • Backward compatibility between versions
    • Wide industry support

    Wi-Fi Standards Evolution

    Standard Year Frequency Max Speed Features
    802.11a 1999 5 GHz 54 Mbps First 5GHz standard, OFDM
    802.11b 1999 2.4 GHz 11 Mbps First widely adopted standard
    802.11g 2003 2.4 GHz 54 Mbps OFDM in 2.4GHz, backward compatible
    802.11n (Wi-Fi 4) 2009 2.4/5 GHz 600 Mbps MIMO, channel bonding
    802.11ac (Wi-Fi 5) 2013 5 GHz 3.5 Gbps MU-MIMO, wider channels
    802.11ax (Wi-Fi 6) 2019 2.4/5/6 GHz 9.6 Gbps OFDMA, TWT, 1024-QAM
    802.11be (Wi-Fi 7) 2024 (est.) 2.4/5/6 GHz 46 Gbps 320 MHz channels, Multi-Link Operation

    Key Technologies in Modern Wi-Fi

    MIMO

    Multiple Input Multiple Output - uses multiple antennas to improve performance

    OFDMA

    Orthogonal Frequency Division Multiple Access - improves efficiency with multiple devices

    Beamforming

    Directs signals toward specific devices rather than broadcasting in all directions

    TWT

    Target Wake Time - improves battery life for IoT devices

    Wi-Fi Frequency Bands

    2.4 GHz Band

    • Frequency range: 2.400 - 2.4835 GHz
    • 14 channels (only 11 usable in US)
    • Better range but more interference
    • Used by many devices (microwaves, Bluetooth)
    • Channel width typically 20 MHz

    5 GHz Band

    • Frequency range: 5.150 - 5.925 GHz
    • 24 non-overlapping channels (more in some regions)
    • Higher speeds but shorter range
    • Less crowded than 2.4 GHz
    • Channel widths up to 160 MHz

    6 GHz Band (Wi-Fi 6E)

    The newest addition to Wi-Fi spectrum (since 2020) offering:

    • Frequency range: 5.925 - 7.125 GHz
    • 59 additional 20MHz channels (or fewer wider channels)
    • No legacy device interference
    • Enables full potential of Wi-Fi 6/6E
    • Requires new hardware (Wi-Fi 6E routers and devices)

    Wi-Fi Network Components

    Access Point (AP)

    Device that allows wireless devices to connect to a wired network using Wi-Fi

    Types: Standalone, Controller-based, Cloud-managed

    Wireless Router

    Combination device that includes AP, router, switch, and often modem functionality

    Features: NAT, firewall, DHCP server

    Wireless Clients

    Devices that connect to the Wi-Fi network (laptops, smartphones, IoT devices)

    Requirements: Wi-Fi adapter, correct drivers

    Wireless Controller

    Centralized management system for multiple APs in enterprise environments

    Functions: Configuration, monitoring, security

    Wi-Fi Network Topologies

    Infrastructure Mode

    Clients connect through an access point (most common)

    Ad-hoc Mode

    Device-to-device direct connection without AP

    Mesh Network

    Multiple APs work together to extend coverage

    WDS (Wireless Distribution System)

    APs connect wirelessly to extend network range

    Wi-Fi Security Protocols

    Protocol Year Encryption Security Level Status
    WEP 1997 RC4 (64/128-bit) Very Weak Deprecated
    WPA 2003 TKIP with RC4 Weak Deprecated
    WPA2 2004 CCMP (AES) Strong Widely Used
    WPA3 2018 GCMP-256 (AES) Very Strong Current Standard

    WPA3 Improvements

    Simultaneous Authentication of Equals (SAE)

    Replaces PSK to prevent offline dictionary attacks

    192-bit Security Suite

    For enterprise networks requiring higher security

    Forward Secrecy

    Prevents decryption of old traffic if password is compromised

    Wi-Fi Security Best Practices

    • Always use WPA3 if available, or WPA2 with AES as fallback
    • Use strong, complex passwords (minimum 12 characters)
    • Disable WPS (Wi-Fi Protected Setup) as it's vulnerable
    • Change default admin credentials on routers/APs
    • Keep firmware updated on all network devices
    • Disable remote administration features
    • Consider using a separate guest network
    • For enterprises, implement 802.1X authentication

    Wi-Fi Performance Optimization

    Channel Selection

    • Use Wi-Fi analyzer tools to find least congested channels
    • In 2.4GHz, use channels 1, 6, or 11 (non-overlapping)
    • For 5GHz, use DFS channels if available and supported
    • Enable automatic channel selection if available

    Router Placement

    • Central location in the area needing coverage
    • Elevated position (not on floor)
    • Away from large metal objects and appliances
    • Orientation of antennas matters (vertical for most devices)

    Advanced Settings

    • Enable band steering to direct clients to 5GHz
    • Adjust transmit power if needed (not always max)
    • Consider wider channels in 5GHz (80MHz or 160MHz)
    • Enable WMM (Wi-Fi Multimedia) for QoS

    Troubleshooting Common Wi-Fi Issues

    Slow Speeds

    • Check for interference
    • Test with wired connection

    Connection Drops

    • Update drivers/firmware
    • Check for overheating

    Dead Zones

    • Consider mesh system
    • Add range extenders

    Authentication Issues

    • Verify password
    • Check security protocol
