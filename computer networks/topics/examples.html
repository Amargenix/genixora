Computer Networks

    Computer Network Examples

    1. Local Area Network (LAN)

    A LAN connects devices within a limited area like a home, office, or building.

    • Example: Office network with connected computers, printers, and servers
    • Technologies: Ethernet, Wi-Fi
    • Typical Speed: 100 Mbps to 10 Gbps
    • Range: Up to 1 km

    2. Wide Area Network (WAN)

    A WAN spans large geographical areas, often connecting multiple LANs.

    • Example: The Internet, corporate networks spanning multiple locations
    • Technologies: MPLS, Frame Relay, leased lines, satellite
    • Typical Speed: 1.5 Mbps to 100 Gbps
    • Range: Unlimited (global)

    3. Metropolitan Area Network (MAN)

    A MAN covers a larger area than a LAN but smaller than a WAN, typically a city.

    • Example: City-wide cable TV network, university campuses with multiple buildings
    • Technologies: Ethernet, ATM, FDDI
    • Typical Speed: 10 Mbps to 10 Gbps
    • Range: 5-50 km

    4. Personal Area Network (PAN)

    A PAN connects devices centered around an individual person's workspace.

    • Example: Bluetooth connection between smartphone and wireless headphones
    • Technologies: Bluetooth, USB, Zigbee, Infrared
    • Typical Speed: 1 Mbps to 50 Mbps
    • Range: Up to 10 meters

    5. Virtual Private Network (VPN)

    A VPN extends a private network across a public network, enabling secure communication.

    • Example: Remote workers accessing company resources securely
    • Technologies: IPsec, SSL/TLS, OpenVPN
    • Typical Use Cases: Secure remote access, bypassing geo-restrictions

    6. Storage Area Network (SAN)

    A SAN provides block-level network access to consolidated storage.

    • Example: Enterprise storage systems accessed by multiple servers
    • Technologies: Fibre Channel, iSCSI, FCoE
    • Typical Speed: 4 Gbps to 32 Gbps (Fibre Channel)

    7. Wireless Local Area Network (WLAN)

    A WLAN allows devices to connect wirelessly within a limited area.

    • Example: Home Wi-Fi network, public hotspots
    • Technologies: Wi-Fi (802.11 standards)
    • Typical Speed: 150 Mbps to 9.6 Gbps (Wi-Fi 6)
    • Range: Up to 100 meters (depending on standard and environment)

    8. Client-Server Network

    A centralized architecture where clients request services from servers.

    • Example: Web browsers (clients) accessing web servers
    • Technologies: HTTP, FTP, SMTP
    • Characteristics: Centralized management, scalable

    9. Peer-to-Peer Network (P2P)

    A decentralized network where each node can act as both client and server.

    • Example: File sharing networks like BitTorrent
    • Technologies: Distributed hash tables (DHT)
    • Characteristics: No central server, resilient to failures

    10. Cloud Computing Networks

    Networks that enable delivery of computing services over the internet.

    • Example: AWS, Azure, Google Cloud Platform
    • Technologies: Virtualization, SDN, APIs
    • Service Models: IaaS, PaaS, SaaS

    Network Topology Examples

    1. Star Topology: All devices connect to a central hub (common in home networks)
    2. Bus Topology: All devices share a single communication line (older Ethernet networks)
    3. Ring Topology: Devices form a circular data path (Token Ring networks)
    4. Mesh Topology: Each device connects to every other device (wireless mesh networks)
    5. Tree Topology: Hierarchical combination of star and bus topologies (large enterprise networks)
