    Internet Control Message Protocol (ICMP)

    ICMP is a network layer protocol used by network devices to diagnose network communication issues and send error messages.

    ICMP Header Format

    Bits 0-7 8-15 16-23 24-31
    Type (8 bits)
    Code (8 bits)
    Checksum (16 bits)
    Message Body (variable)

    Common ICMP Message Types

    Type Code Description
    0 0 Echo Reply (Ping response)
    3 0-15 Destination Unreachable
    4 0 Source Quench (Congestion control)
    5 0-3 Redirect Message
    8 0 Echo Request (Ping)
    11 0-1 Time Exceeded
    12 0-1 Parameter Problem

    Key Functions of ICMP

    • Error Reporting: ICMP reports errors encountered during IP packet processing
    • Network Diagnostics: Tools like ping and traceroute use ICMP
    • Flow Control: Source Quench messages help manage congestion
    • Route Optimization: Redirect messages help hosts find better routes
    • Reachability Testing: Echo request/reply (ping) verifies host availability

    ICMP vs. TCP/UDP

    Feature ICMP TCP/UDP
    Layer Network (3) Transport (4)
    Reliability Unreliable TCP: Reliable, UDP: Unreliable
    Port Numbers No ports Uses ports
    Primary Use Error reporting, diagnostics Data transmission

    Common ICMP Tools

    Ping

    Uses ICMP Echo Request (Type 8) and Echo Reply (Type 0) messages to test host reachability and measure round-trip time.

    ping www.example.com

    Traceroute/Tracert

    Uses ICMP Time Exceeded (Type 11) messages to map the path packets take to reach a destination.

    traceroute www.example.com  # Linux/Mac
                tracert www.example.com    # Windows

    ICMP Security Considerations

    • ICMP Flood Attacks: Can be used in DDoS attacks by overwhelming targets with ICMP packets
    • Ping of Death: Sending malformed or oversized packets to crash systems
    • Smurf Attack: Spoofed ICMP broadcast requests that amplify traffic
    • Information Disclosure: ICMP messages may reveal network topology information

    Many networks implement ICMP rate limiting or filtering to mitigate these risks while maintaining useful functionality.

    ICMPv6 Enhancements

    ICMPv6 (used with IPv6) includes additional functionality beyond ICMPv4:

    • Neighbor Discovery Protocol (replaces ARP)
    • Multicast Listener Discovery
    • Path MTU Discovery
    • Router Solicitation and Advertisement

    Summary of ICMP Characteristics

    • Operates at the Network Layer (Layer 3)
    • Protocol number 1 (for IPv4)
    • Doesn't use ports like TCP/UDP
    • Essential for network diagnostics and error reporting
    • Used by common network tools (ping, traceroute)
    • Both IPv4 (ICMP) and IPv6 (ICMPv6) versions exist
