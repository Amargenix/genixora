    Network Security in Computer Networks

    Introduction to Network Security

    Network security encompasses the policies, technologies, and practices designed to protect network infrastructure and data from unauthorized access, misuse, modification, or denial of service.

    CIA Triad

    • Confidentiality: Protecting data from unauthorized access
    • Integrity: Ensuring data accuracy and consistency
    • Availability: Ensuring systems are accessible when needed

    Security Objectives

    • Prevention of unauthorized access
    • Detection of intrusions and anomalies
    • Response to security incidents
    • Recovery from attacks

    Common Network Security Threats

    Threat Type Description Examples Impact
    Malware Malicious software designed to harm systems Viruses, worms, trojans, ransomware Data loss, system damage
    Phishing Deceptive attempts to steal sensitive information Email scams, fake websites Credential theft, financial loss
    DDoS Overwhelming network resources with traffic Botnet attacks, SYN floods Service disruption
    MITM Intercepting communication between parties Session hijacking, Wi-Fi eavesdropping Data interception
    SQL Injection Inserting malicious code into database queries Web form exploitation Data breach
    Zero-Day Exploiting unknown vulnerabilities Advanced persistent threats System compromise

    Network Security Technologies

    Firewalls

    • Packet-filtering: Examines headers
    • Stateful inspection: Tracks connections
    • Next-gen firewalls: Deep packet inspection
    • Web application: Protects web apps

    Intrusion Systems

    • IDS: Intrusion Detection System
    • IPS: Intrusion Prevention System
    • SIEM: Security Information and Event Management
    • Behavioral analysis: Anomaly detection

    Encryption

    • SSL/TLS: Secure web traffic
    • IPSec: Secure network layer
    • VPN: Secure remote access
    • E2EE: End-to-end encryption

    Access Control

    • RBAC: Role-Based Access Control
    • ABAC: Attribute-Based Access Control
    • MFA: Multi-Factor Authentication
    • IAM: Identity and Access Management

    Network Segmentation

    • VLANs: Virtual LANs
    • Microsegmentation: Granular isolation
    • Zero Trust: Verify everything
    • Air gaps: Physical separation

    Other Technologies

    • Honeypots: Decoy systems
    • DLP: Data Loss Prevention
    • NAC: Network Access Control
    • Sandboxing: Safe execution

    Network Security Protocols

    Transport Layer Security

    SSL/TLS: Cryptographic protocols for secure communication

    • Version evolution: SSL 1.0-3.0 → TLS 1.0-1.3
    • Uses asymmetric cryptography for key exchange
    • Provides encryption, authentication, integrity

    IP Security

    IPSec: Secure Internet Protocol communications

    • Operates at network layer (Layer 3)
    • Two modes: Transport (host-to-host) and Tunnel (network-to-network)
    • Components: AH (authentication), ESP (encryption), IKE (key management)
