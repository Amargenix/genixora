Computer Networks

    Wide Area Network (WAN)

    A Wide Area Network (WAN) is a telecommunications network that extends over a large geographical area for the primary purpose of computer networking.

    Key Characteristics of WAN

    • Geographical Coverage: Spans large distances (countries, continents, or worldwide)
    • Infrastructure: Uses leased telecommunication lines or satellite links
    • Speed: Generally slower than LANs due to longer distances
    • Ownership: Typically owned by service providers, not end users
    • Cost: Higher setup and maintenance costs compared to LANs

    WAN Technologies

    Circuit Switching

    Establishes a dedicated communication path before transmission begins.

    Examples: PSTN (Public Switched Telephone Network), ISDN

    Packet Switching

    Data is broken into packets and sent independently over the network.

    Examples: X.25, Frame Relay, ATM (Asynchronous Transfer Mode)

    Internet-based WAN

    Uses internet protocols and infrastructure for WAN connectivity.

    Examples: MPLS, VPN, SD-WAN

    Common WAN Connection Types

    Technology Speed Description
    Leased Lines 56 Kbps - 10 Gbps Dedicated point-to-point connection
    DSL 256 Kbps - 100 Mbps Digital Subscriber Line over telephone lines
    Cable Modem 10 Mbps - 1 Gbps Uses cable television infrastructure
    Fiber Optic 100 Mbps - 100 Gbps High-speed using light pulses through glass fibers
    Satellite 1 Mbps - 100 Mbps Wireless connection via communication satellites

    WAN Protocols

    PPP (Point-to-Point Protocol)

    A data link protocol used to establish a direct connection between two nodes. It provides authentication, encryption, and compression.

    HDLC (High-Level Data Link Control)

    A bit-oriented synchronous data link layer protocol developed by ISO. It's the default encapsulation type on most Cisco routers.

    Frame Relay

    A packet-switching protocol for connecting devices on a WAN. It operates at the data link layer and is more efficient than X.25.

    WAN vs LAN Comparison

    Feature WAN LAN
    Coverage Area Large geographical area Small local area
    Speed Slower (Mbps-Gbps) Faster (Gbps-Tbps)
    Cost Higher Lower
    Ownership Service providers Individual organizations
    Technologies MPLS, Frame Relay, ATM Ethernet, Wi-Fi

    Emerging WAN Technologies

    SD-WAN (Software-Defined WAN)

    Uses software to control the connectivity, management, and services between data centers and remote branches. Benefits include:

    • Improved application performance
    • Reduced costs by using multiple connection types
    • Simplified management through centralized control
    • Increased agility and flexibility

    5G WAN

    The integration of 5G cellular technology into WAN architectures offers:

    • High-speed wireless connectivity (up to 10 Gbps)
    • Low latency (1ms or less)
    • Massive device connectivity

    WAN Security Considerations

    Due to their public nature, WANs require special security measures:

    • Encryption: VPNs, IPsec, SSL/TLS to protect data in transit
    • Firewalls: To filter incoming and outgoing traffic
    • Authentication: Strong user and device authentication
    • Intrusion Detection/Prevention: Monitoring for malicious activity
    • Access Control: Strict policies on who can access network resources
