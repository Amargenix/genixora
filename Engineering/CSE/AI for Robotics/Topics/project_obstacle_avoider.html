    3. Avoidance Algorithms

    Approaches

    • Bug Algorithms: Simple wall following
    • Potential Fields: Attraction/repulsion
    • Dynamic Window: Velocity space search
    • RL-based: Learned avoidance

    Algorithm Comparison

    Algorithm Complexity Best For
    Bug Low Simple environments
    Potential Field Medium Dynamic obstacles
    DWA High High-speed navigation
    Tip: Dynamic Window Approach (DWA) offers best balance for most applications
    # Potential Field pseudocode
    def calculate_force(robot_pos, goal_pos, obstacles):
      # Attractive force to goal
      att_force = k_att * (goal_pos - robot_pos)
      
      # Repulsive forces from obstacles
      rep_force = Vector(0, 0)
      for obs in obstacles:
        dist = distance(robot_pos, obs.position)
        if dist < obs.influence_radius:
          direction = (robot_pos - obs.position).normalized()
          rep_force += k_rep * (1/dist - 1/obs.influence_radius) * direction

      return att_force + rep_force
